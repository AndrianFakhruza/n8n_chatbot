Sistem Chatbot RAG Cerdas

Implementasi AI Automation untuk Respon WhatsApp Otomatis

Konsep Proyek (RAG Architecture)

Proyek ini adalah implementasi Retrieval-Augmented Generation (RAG). Tujuannya adalah memastikan chatbot dapat memberikan jawaban yang tidak hanya kreatif, tetapi juga didukung oleh fakta dan data spesifik yang tersimpan dalam database eksternal. Ini mengatasi keterbatasan pengetahuan model AI pada batas waktu pelatihan mereka.

Alur Kerja (Workflow) pada n8n

1. Trigger: Pesan baru masuk melalui WhatsApp Trigger.

2. Orkesrasi (AI Agent): Pesan diteruskan ke AI Agent (n8n) yang bertindak sebagai pengatur alur cerdas.

3. Basis Pengetahuan: Jika Agen membutuhkan informasi, ia menggunakan Supabase Vector Store sebagai tool untuk mencari data kontekstual.

4. Respon: Gemini merangkum hasil pencarian dan konteks percakapan menjadi balasan, yang kemudian dikirimkan kembali ke pengguna WhatsApp.

Komponen Teknis (Technology Stack)

  • Orkestrasi: n8n - Mengelola seluruh workflow otomatisasi dan integrasi antar layanan.
  • Model Chat: Google Gemini Chat Model - Inti kecerdasan buatan untuk penalaran dan pembuatan respons.
  • Memori Percakapan: Postgres Chat Memory - Menyimpan riwayat obrolan di PostgreSQL untuk konteks yang berkelanjutan.
  • Vector Store (RAG): Supabase Vector Store - Basis data eksternal menggunakan Supabase dengan extension pgvector untuk pencarian semantik.
  • Embedding: Embeddings Google Gemini - Mengubah teks (dokumen dan query pengguna) menjadi vektor numerik yang dapat dicari.
